

**【角色设定与技术栈】**
你现在是一名顶级的资深前端UI工程师。请使用 **Vue3 (Composition API) + Tailwind CSS** 为我开发一个“晨读刮刮乐”应用的首屏欢迎弹窗组件。

**【设计目标与场景适配】**
这个页面将主要展示在**学校多媒体教室的触控大屏**上（通常为1080p或4k分辨率的16:9屏幕），因此**所有的UI元素、字体必须非常巨大、醒目，且具备极强的视觉冲击力和互动感**，以吸引小学生的注意力。

**【核心UI结构与详细样式要求】**

**1. 全局遮罩层 (Backdrop)**
*   实现一个全屏的绝对定位遮罩层，覆盖整个视口，让底层内容模糊。
*   **毛玻璃效果：** 使用深度模糊效果（类似于 `backdrop-blur-xl` 或自定义更大的 blur 值），背景颜色使用带有一定透明度的暗色（如 `bg-black/40` 或带有微弱暖色调的半透明遮罩），确保弹窗内容在任何复杂背景下都能清晰凸显。
*   布局：使用 Flexbox，将内部的内容容器完全垂直和水平居中。

**2. 核心标题 (Title)**
*   **文案：** `👏 有请代表抽取今日奖励 👏` （左右两侧加上鼓掌或点赞的emoji作为装饰，参考图片中的黄色块状物）。
*   **字号与排版：** 字体极其巨大（在 Tailwind 中可能需要使用 `text-6xl`, `text-7xl` 甚至自定义更大的如 `text-[80px]` 配合较粗的字重 `font-extrabold`）。
*   **色彩与特效：**
    *   文字本身使用从上到下的线性渐变色（顶部为明亮的浅黄色，底部为深金黄色/橘黄色）。
    *   必须带有强烈的文字描边或发光阴影效果（Text Shadow / Text Stroke）。建议使用深褐色或橘红色的多重文字阴影，让黄色文字呈现出强烈的3D立体感和描边感，绝对不能和背景融为一体。

**3. 互动大按钮 (Main Call-to-Action Button)**
*   **文案：** `点击抽取今日奖励 👈`（文案后面加上一个手指引导的图标）。
*   **尺寸与形状：** 按钮极度宽大，采用完全的药丸形状（`rounded-full`），高度至少在 `100px` 以上，宽度根据文字自适应并保留极宽的左右 padding（如 `px-16 py-6`）。
*   **按钮内部色彩：** 采用强烈的垂直渐变，顶部为亮黄色（类似 `#ffe100`），底部过渡到鲜艳的橘红色（类似 `#ff5500`）。
*   **文字样式：** 按钮内的文字必须非常大（如 `text-5xl` `font-bold`），文字颜色为白色，并且必须带有黑色的文字阴影或描边，确保在亮色渐变背景上依然极度清晰。
*   **✨ 重点特效：金色流动边框动画 ✨**
    *   在按钮的外围，实现一圈流动、发光的金色光彩特效。
    *   **技术实现建议（请AI严格参考）：** 建议不要直接操作 border。可以使用一个相对定位的包裹容器。在按钮底层放一个稍微大一点的绝对定位的伪元素或 div，给它应用一个带有金色和透明相间的 `conic-gradient`（圆锥渐变），并使用 CSS `@keyframes` 让这个渐变层无限旋转（`spin`）。然后将真正的按钮（带有实色背景）覆盖在上面，从而露出一圈不断流动的金色边缘。
    *   在这个流动边框的外部，还可以加一层轻微的黄色外发光（`box-shadow: 0 0 20px #fbbf24`），增强“宝箱发光”般的视觉体验。

**4. 交互动效 (Interactions)**
*   弹窗出现时，整体带有一个弹性放大的入场动画（Scale up with a spring effect）。
*   按钮在 Hover 时，带有轻微的放大效果（`hover:scale-105`），流动边框的旋转速度可以稍微加快。
*   按钮在 Active（点击）时，有明显的下压反馈（`active:scale-95`）。
